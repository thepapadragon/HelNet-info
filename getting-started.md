# Getting Started

Make sure the bot has the necessary permissions for the channels you want it to send messages in.

Commands:
* `/setup server`
  * Server Color - This changes accent colors on embeds to match your server theme
  * More soon!
* `/setup modules` - Turn features on (All are off by default)
  * [Time Module](time-module.md) - Access to the Time terminal and Time suggestion commands
  * [Division Module](division-module.md) - Create divisions and a terminal for your members to join
  * [Announcement Module](announcement-module.md) - Create themed announcement presets to use at any time
