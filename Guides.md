# Guides

## Getting started
Make sure the bot has the necessary permissions for the channels you want it to send messages in
Commands:
* `/setup server`
  * Server Color - This changes accent colors on embeds to match your server theme
  * More soon!
* `/setup modules` - Turn features on (All are off by default)
  * Time Module - access to the Time terminal and Time suggestion commands
  * Division Module - Create divisions and a terminal for your members to join
  * Announcement Module - Create themed announcement presets to use at any time

## Time Module
Access to the Time terminal and Time suggestion commands
### Time Terminal
This is a terminal you deploy into a channel. Users can use this terminal to set their time zone preferences (UTC offset and whether their region observes Daylight Saving Time/Summer Time). The terminal then displays all the time zones your server consists of and the current time in each.
### Time Suggestions
Only people with their time preferences set can use this. There are 2 ways to use this:
* Method 1 - When suggesting a time, include "!" right before the suggestion. Example: `Let's play at !5pm`
* Method 2 - Find a message with a time in it. Example: `Let's play at 5pm` -> Long press/right click -> apps -> HelNet -> Time Suggest (The original sender must have their time preferences set)
